CHOIX TECHNIQUE
---------------
- PWA (Progressive Web App) avec Next.js 14 + TypeScript
- Installable sur iPhone ET Android depuis le navigateur (Chrome/Safari)
- 100% hors ligne apres premier chargement (Service Worker)
- Stockage local : IndexedDB via Dexie.js
- OCR : Tesseract.js (tourne dans le navigateur, gratuit, pas de backend)
- Graphiques : Recharts
- Style : Tailwind CSS
- Pas de serveur, pas de base de donnees distante, tout en local

CE QUI RESTE A FAIRE
---------------------

1. GENERER LES ICONES (5 min)
   Les icones PNG sont necessaires pour que la PWA soit installable.
   Commande : node scripts/generate-icons.js
   Cree : public/icon-192.png et public/icon-512.png

2. INSTALLER LES DEPENDANCES (2 min)
   Dans le dossier pwa/ :
     npm install

   Si erreur avec next-pwa :
     npm install --legacy-peer-deps

3. LANCER EN DEV (1 min)
   npm run dev
   Ouvrir : http://localhost:3000

4. TESTER L'OCR
   - Sur mobile : utiliser l'appareil photo pour prendre un ticket
   - Sur desktop : uploader une photo de ticket
   - Le modele Tesseract se telecharge au premier usage (~10MB)

5. TESTER LES NOTIFICATIONS
   - Definir un budget dans l'onglet Budget
   - Ajouter des depenses jusqu'a 80% du budget
   - Accepter les permissions de notification quand demande

6. BUILD PRODUCTION (optionnel - pour heberger)
   npm run build
   Genere le dossier "out/" - fichiers statiques a deployer sur :
   - GitHub Pages (gratuit)
   - Netlify (gratuit)
   - Vercel (gratuit)
   - Ou serveur web personnel

7. INSTALLER SUR TELEPHONE
   Android (Chrome) :
     Aller sur l'URL → menu ⋮ → "Installer l'application"

   iPhone (Safari) :
     Aller sur l'URL → bouton partager → "Sur l'ecran d'accueil"

   Note : sur iPhone, les notifications ne fonctionnent que si
   l'app est ajoutee a l'ecran d'accueil (PWA standalone mode).
